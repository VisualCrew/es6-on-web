# ES6 ON WEB

玩遍 ECMAScript 2015



### 使用方法

  ```
  git clone
  cd
  npm install
  npm start
  ```

之后，打开浏览器（[Chrome](http://www.google.cn/intl/zh-CN/chrome/browser/)），访问：```http://localhost:8080/``` 查看控制台即可；

如果只需要测试某个模块的用法，可以打开[]()文件，注释掉其他模块的导入即可；
